🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════

ADMIN (Authenticated):
  GET    /api/admin/faqs              → Get all FAQs
  POST   /api/admin/faqs              → Create FAQ
  PUT    /api/admin/faqs/[id]         → Update FAQ
  DELETE /api/admin/faqs/[id]         → Delete FAQ

PUBLIC (Unauthenticated):
  GET    /api/faqs                    → Get active FAQs (sorted)
